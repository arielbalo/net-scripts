# Script para enumerar una red y los puertos de una computadora

## Python 3
```
usage: python3 netenum.py [-h] [-r RANGE] [-ip IPADDRESS] [-t TIMEOUT] [-v]

options:
  -h, --help                              |  show this help message and exit
  -r RANGE, --range RANGE                 |  IP range to scan (Ex. '192.168.0.0/24')
  -ip IPADDRESS, --ipaddress IPADDRESS    |  IP address to port scan (Ex. '192.168.0.100')
  -t TIMEOUT, --timeout TIMEOUT           |  Timeout in seconds (Default: 2)
  -v, --verbose                           |  Show all the process
```
